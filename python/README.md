making a movie recommendation program

The movie recommendation system aim is to provide users with personalized movie recommendations based on their preferred genres. 

Users can input a preferred movie genre, and the system will provide a list of recommended movies within that genre. 

Users have the option to contribute to the movie database by adding new movies, enhancing the system's flexibility and inclusiveness. 

Movies data is stored in a CSV file, ensuring that user-contributed movies and recommendations are saved for future interactions.
